# Source2Target Documentation

This directory contains all project documentation organized by category.

## Directory Structure

### `/docs/` (root) - User-Facing Guides
- **USER_GUIDE.md** - End-user guide for mapping workflow
- **ADMIN_GUIDE.md** - Administrator configuration guide
- **DEVELOPER_GUIDE.md** - Developer setup and contribution guide
- **QUICK_START.md** - Quick start guide for new users
- **TRANSFORMATION_MANAGEMENT.md** - Transformation library management

### `/docs/architecture/` - System Architecture
- **V2_SCHEMA_DIAGRAM.md** - Database schema and data flow diagrams
- **V2_MULTI_FIELD_VECTOR_SEARCH.md** - AI/ML vector search architecture
- **ADMIN_INTERFACE_STRUCTURE.md** - Admin interface design

### `/docs/implementation/` - Implementation Details
- **V2_IMPLEMENTATION_PLAN.md** - V2 implementation roadmap
- **V2_BACKEND_COMPLETE.md** - Backend implementation status
- **V2_BULK_UPLOAD_AND_JOINS.md** - Bulk upload feature design
- **V2_LIVE_MAPPED_FIELDS.md** - Live mapped fields feature
- **AI_SEARCH_REAL_API_INTEGRATION.md** - AI search integration details

### `/docs/changelog/` - Change History & Bug Fixes
- **UI_IMPROVEMENTS_SUMMARY.md** - UI/UX improvements log
- **MAPPING_UPDATE_ALL_FIXES.md** - Mapping CRUD bug fixes
- **DELETE_MAPPING_FIX.md** - Delete mapping bug fix
- **TARGET_DESCRIPTION_SETUP.md** - Target description feature
- **MATCH_QUALITY_THRESHOLDS.md** - Match quality scoring updates
- **HELP_BUTTONS_UPDATE.md** - Help system improvements
- **DOCUMENTATION_UPDATE_SUMMARY.md** - Documentation updates
- (and other fix/update logs)

### `/docs/deployment/` - Deployment & Operations
- **V2_DEPLOYMENT_CHECKLIST.md** - Production deployment checklist
- **DATABRICKS_DEPLOY.md** - Databricks-specific deployment
- **V2_SUMMARY.md** - V2 executive summary

### `/database/` - Database Files (separate directory)
Database-specific files remain in `/database/`:
- Migration scripts (SQL)
- Database documentation (V2_MIGRATION_GUIDE.md, etc.)
- Constraints documentation
