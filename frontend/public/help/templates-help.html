How to prepare and upload your source fields for AI mapping

üìã Why Upload Source Fields?

Smart Mapper V4 uses a Target-First workflow. Before AI can suggest mappings, it needs to know all your available source fields.

The AI uses your source data to:

  • Find semantically similar columns for each target field
  • Match column descriptions to historical patterns
  • Identify join keys and filter columns
  • Generate accurate SQL expressions
‚ö†Ô∏è Quality In = Quality Out

The better your source field descriptions, the better the AI suggestions. Take time to write clear, meaningful descriptions in the src_comments column.

üìÑ CSV Template Format

Download the template from the project page, then fill it with your source field information.

Column Type Description Example

üí° Pro Tip: Good Descriptions

Instead of just "ID", write "Unique identifier for the recipient in the base table". The more context, the better the AI matching!

üì§ Upload Process

  1. Open your project from the Projects page
  2. Click üì§ Upload Sources
  3. Click "Download Template" to get the CSV format
  4. Fill in your source tables and columns
  5. Save the CSV file
  6. Click "Select CSV File" and choose your file
  7. Click "Upload"

What Happens During Upload

  • File is validated for required columns
  • Rows are checked for completeness
  • Source fields are inserted into unmapped_fields table
  • Semantic fields are generated for vector search
  • Progress shows during upload (may take a minute for large files)
‚ö†Ô∏è Upload Size

Large uploads (100+ fields) may take 1-2 minutes. The progress indicator shows status - don't close the page while uploading.

üîÑ Adding More Fields Later

You can upload additional source fields at any time:

  • New fields are added to existing ones (not replaced)
  • Duplicates (same table + column) are detected and skipped
  • After upload, you may need to Regenerate suggestions for tables that need the new fields

‚ùì Troubleshooting

Upload Fails

  • Check that all required columns are present in your CSV
  • Ensure column headers match exactly (case-sensitive)
  • Verify CSV is properly formatted (no extra commas, quotes are escaped)

AI Suggestions Are Poor

  • Check that src_comments has meaningful descriptions
  • Ensure you've uploaded ALL relevant source tables
  • Try regenerating individual suggestions

Fields Not Showing in Suggestions

  • Verify the field was successfully uploaded (check project detail page)
  • Make sure the project_id matches
  • Regenerate suggestions for the target table
