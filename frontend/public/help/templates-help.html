üéØ Platform Overview

The Gainwell Source-to-Target Mapping Platform is an enterprise-grade healthcare data integration solution that leverages artificial intelligence to automate the complex process of mapping source database fields to target system schemas.

Healthcare Focus: Specifically designed for Medicaid agencies, healthcare providers, and public health organizations modernizing their data infrastructure.

üöÄ Getting Started Workflow

1

Enterprise Configuration Setup

Navigate to the Configuration section to establish connections to your healthcare data warehouse. Configure your Databricks workspace, set up AI model endpoints, and validate connections to your Medicaid or healthcare management systems.

üîß View Configuration Demo
2

Field Mapping Analysis

Access the Field Mapping interface with dual-tab navigation: Unmapped Fields for source columns requiring mapping, and Mapped Fields for reviewing completed field relationships. Use search, filtering, and export capabilities to manage your mapping project efficiently.

üìä View Field Mapping Demo
3

AI-Powered Mapping Generation

In the Unmapped Fields tab, select any source field to receive AI-generated mapping suggestions. The system uses semantic similarity and healthcare-trained models to suggest appropriate target fields with detailed reasoning and confidence scores.

4

Mapping Review and Approval

Review AI suggestions with detailed reasoning before approval. Select the most appropriate mapping and save it to your enterprise systems. Approved mappings immediately appear in the Mapped Fields tab for ongoing management.

5

Mapping Management and Analytics

Use the Mapped Fields tab to view all completed mappings with statistics, search functionality, and export capabilities. Monitor progress with real-time metrics showing total mapped fields, source tables, and target tables covered.

6

Data Export and Documentation

Export mapping data in CSV format with timestamps for documentation and audit purposes. Use search filters to create targeted exports for specific tables, columns, or mapping relationships.
