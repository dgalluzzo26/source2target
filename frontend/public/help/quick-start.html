1

üìÅ Create a Project

  1. Click + New Project from the Projects page
  2. Enter a Project Name (e.g., "DMES Member Migration")
  3. Add optional Description and Source System
  4. Set Target Domains to filter which tables to map (e.g., "Member")
  5. Click Create Project
üí° Tip: Add team member emails to collaborate on the project!

3

ü§ñ Start AI Discovery

Let the AI analyze your sources and suggest mappings:

  1. From the project page, find the target table you want to map
  2. Click the Discover button
  3. Watch progress: "Discovering 5/25..." updates in real-time
  4. Wait for status to change to SUGGESTIONS_READY

What happens during discovery:

  • AI searches for historical mapping patterns
  • Vector search finds semantically similar source columns
  • LLM rewrites SQL with your source columns
  • Confidence scores are calculated
üí° Tip: Discovery takes 1-5 minutes depending on table size. You can work on other tables while waiting!
