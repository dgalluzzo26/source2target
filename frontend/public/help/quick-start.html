Let AI find the best matches:

Evaluate the AI suggestions:

  • Review the suggestions table showing:

    • Target Table and Target Column
    • Confidence Score - Higher is better (0.8+ is excellent!)
    • Reasoning - Why this match was suggested
  • Click Select Target on the best match
  • The mapping is saved automatically!

‚ùì Common Questions

Q: How long does AI take?
A: Usually 15-25 seconds for vector search and suggestions generation.

Q: Can I change a mapping later?
A: Yes! Delete the old mapping from Mapped Fields and create a new one.

Q: What if no AI suggestions appear?
A: Check system status to ensure Vector Search and AI Model are ready. If they are, try manual search instead.

Q: How many fields should I map at once?
A: Work at your own pace. Some users map 10-20 fields per session, others prefer to complete an entire table.
